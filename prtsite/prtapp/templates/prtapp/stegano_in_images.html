Описание метода псевдослучайного интервала для сокрытия информации в изображении

Рассмотрим сокрытие данных в пространственной области. Алгоритмы сокрытия, использующие данные методы, внедряют секретную информацию в пространственные области исходного изображения.

Цветное изображение С будем представлять через дискретную функцию, которая определяет вектор цвета с(x, y) для каждого пикселя изображения (x, y), где x и y - координаты пикселя, а значение цвета задаёт трехкомпонентный вектор в цветовом пространстве. Наиболее распространённый способ передачи цвета – это модель RGB, в которой основные цвета красный, зеленый, синий, а остальные могут быть представлены в виде взвешенной суммы остальных цветов.

Вектор цвета с(x, y) в RGB-пространстве представляет интенсивность основных цветов. Сообщение встраиваются за счёт манипуляций цветовыми составляющими {R(x,y), G(x,y), B(x,y)} или непосредственно яркостью.

Общий принцип этих методов заключается в замене избыточной, малозначимой части изображения битами секретного сообщения. Для извлечения сообщения необходимо знать алгоритм, по которому размещалась по контейнеру скрытая информация.

В данной реализации метода псевдослучайного интервала рассматривается генератор вида – 1, 2, 3, 4. Это означает, что кодированная информация содержится в первом пискеле, потом через один пиксель, через два, через три, через четыре. Далее генератор повторяет этот ряд.

Рассматривается синий цветовой компонент изображения. Так как в каждом закодированном пикселе кодируются только 2 наименее значащих бита, то метод эффективен в случае, когда битовая длина секретного сообщения существенно меньше количества пикселей изображения.

Отметим, что в данной реализации рассматриваются в качестве контейнера изображения в формате BMP.
